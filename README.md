# Middle Earth Registry

Next.js loyihasi - Foydalanuvchilarni qo'shish va o'chirish tizimi.

## Ishga tushirish

### Local development

```bash
npm install
npm run dev
```

Brauzerda `http://localhost:3000` ochiladi.

### Vercel deploy

1. GitHub/GitLab ga push qiling
2. Vercel.com ga kiring
3. Loyihani import qiling
4. Deploy tugmasini bosing
